

 This folder contains the following samples:



  Decoding
  ~~~~~~~~

  This VisualBasic sample demonstrates how to decode JPEG2000 files via
  J2K-Codec's API and how to display them fast in VB.


  Video_even_odd
  ~~~~~~~~~~~~~~

  This VisualBasic sample will be useful for live-video system developers
  who get their frames in two parts - odd and even fields separately.
  
  This sample shows an easy way to combine them using J2K-Codec's "buffer_pitch"
  parameter.
